# 请求身份

说明

为开发者分配Brand（品牌） AccessKey（开发者标识，确保唯一）和 SecretKey（用于接口加密，确保不易被穷举，生成算法不易被猜测）。

# 防止篡改

说明

brand + timestamp + signature 方案

brand: 品牌（开放平台给予的第三方平台的品牌代码） timestamp: 时 间 戳 Unix timestamp（ 毫 秒 ） signature：生成签名

# http 请求头（Http Header）
名称 类型 必填 说明 备注
brand String Y 设备厂家品牌 开放平台给予的第三方平台的品牌代码
timestamp String Y 时间戳 签名时使用的时间戳
signature String Y 签名 生成签名

参考工具类（Java 版）

package com.shimao.iot.elkserver.utils;

import com.alibaba.fastjson.JSON;
import com.alibaba.fastjson.TypeReference;
import org.apache.commons.codec.digest.DigestUtils;

import java.io.UnsupportedEncodingException;
import java.net.URLEncoder;
import java.nio.charset.Charset;
import java.nio.charset.StandardCharsets;
import java.text.MessageFormat;
import java.util.*;

/**
 * @author bka
 * @date 2019-07-31 11:13
 */
public class CertificateUtils {

    private final static Charset CHARSET = StandardCharsets.UTF_8;

    /**
     * 创建签名
     * 注意：外部不要将timestamp、accessKey和sign这4个参数添加到方法参数params集合中
     * params就是请求的参数
     */
    public static String createSign(final HashMap<String, String> params, String accessKey, String secretKey, String timestamp) {
        //1、除去空值请求参数
        HashMap<String, String> newRequestParams = removeEmptyParam(params);

        //2、按照请求参数名的字母升序排列非空请求参数（包含AccessKey,Timestamp)，使用URL键值对的格式（即key1=value1&key2=value2…）拼接成字符串
        newRequestParams.put("AccessKey", accessKey);
        newRequestParams.put("Timestamp", timestamp);
        String strUrlParams = parseUrlString(newRequestParams);

        //4、最后拼接上secretKey得到字符串stringSignTemp
        String stringSignTemp = MessageFormat.format("{0}&SecretKey={1}", strUrlParams, secretKey);

        //5、对stringSignTemp进行MD5运算，并将得到的字符串所有字符转换为大写，得到sign值
        return DigestUtils.md5Hex(stringSignTemp.getBytes(CHARSET)).toUpperCase();
    }

    /**
     * 移除空请求参数
     */
    private static HashMap<String, String> removeEmptyParam(final HashMap<String, String> params) {
        HashMap<String, String> newParams = new HashMap<>(10);
        if (params == null || params.size() <= 0) {
            return newParams;
        }
        for (String key : params.keySet()) {
            String value = params.get(key);
            if (value == null || "".equals(value)) {
                continue;
            }
            newParams.put(key, value);
        }
        return newParams;
    }

    /**
     * 使用URL键值对的格式（即key1=value1&key2=value2…）将参数列表拼接成字符串
     */
    private static String parseUrlString(final HashMap<String, String> requestMap) {
        List<String> keyList = new ArrayList<>(requestMap.keySet());
        Collections.sort(keyList);

        List<String> entryList = new ArrayList<>();
        for (String key : keyList) {
            String value = requestMap.get(key);
            try {
                value = URLEncoder.encode(value, CHARSET.name());
            } catch (UnsupportedEncodingException e) {
                e.printStackTrace();
            }
            entryList.add(MessageFormat.format("{0}={1}", key, value));
        }
        return String.join("&", entryList);
    }

    /**
     * 测试main 方法
     *
     * @param args
     */
    public static void main(String[] args) {
        // Body
        String params = "{\"spaceId\":2000014}";
        HashMap<String, String> hashMap = JSON.parseObject(Objects.requireNonNull(params), new TypeReference<HashMap<String, String>>() {
        });
        String timestamp = String.valueOf(System.currentTimeMillis());
        String sign = createSign(hashMap, "shimao62ae1634e0b32d0", "f6339f4974cac548080ecdeca4037489", timestamp);
        System.out.println("sign : " + sign);
        System.out.println("timeMillis : " + timestamp);
    }

}

参数签名

  1. 按照请求参数名的字母升序排列非空请求参数（包含 AccessKey 与 Timestamp），使用 URL 键值对的格式（即key1=value1&key2=value2…）拼接成字符串 stringA；

  2. 在 stringA 最后拼接上 SecretKey 得到字符串stringSignTemp；

  3. 对 stringSignTemp 进行 MD5 运算，并将得到的字符串所有字符转换为大写，得到

    sign 值。

注（Timestamp 与协议头的timestamp 需要一致）

请求头携带参数(brand, timestamp, signature)，只有拥有合法的身份 brand 和正确的签名 sign 才能放行。

接口类型 HTTP
HTTP Method POST
URI /api/simolink/v2/device/register

# 请求参数

参数名 参数类型 参数说明 是否必填
deviceCode String 设备唯一ID 是
registerType String 注册场景类型 否 home：家居场景，返回的clientId 以home_开头 hotel：酒店场景，返回的clientId 以hotel_开头

# 请求示例

{
   "deviceCode": "ABC123XXX", 
   "registerType": "home"
}

# 响应结果
参数名 参数类型 参数说明 是否必填
retcode int 返回状态码 是
msg String 返回信息 是
data JSON Object 返回业务数据 是
└clientId String 分配的 MQTT clientId
└connection JSON Object MQTT 连接信息 是
└└ip String MQTT broker 连接地址 是
└└port int MQTT broker 连接端口 是
└└username String MQTT 连接认证用户名 是
└└password String MQTT 连接认证密码 是
└└keepAlive int MQTT 连接心跳间隔， 单位：秒 是
└topics JSON Object 订阅/发布主题 是
└└pub String Array 发布主题 是
└└sub String Array 订阅主题 是
└ntpConfig JSON Object ntp配置信息 是
└└ntpAddress String ntp服务器地址 是
└└cycle String ntp同步时间周期 是
└└timeUint String ntp同步时间单位(MINUTE: 分钟, SECOND:秒) 是
