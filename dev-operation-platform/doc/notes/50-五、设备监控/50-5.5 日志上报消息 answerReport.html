    # 日志上报消息 answerReport

    发起端：终端设备

    鉴权方式：参照《1.2.1 签名方式说明》

    HTTP Method POST

    请求类型: application/json

    URL: /device-monitor/v2/log/report

    # 请求参数说明

    参数名 参数类型 参数说明 是否必填
    deviceCode String 设备唯一ID 是
    requestId String 推 送 消 息 中 的 requestId 是
    pushStatus int 上报状态，2 已收到 4 其他错误 是
    errorMessage String 错误说明 否

    # 请求示例

    {
  "deviceCode": "112333eee",
  "requestId": "84986255ce444a1bb2f1ffe9a8464a19",
  "pushStatus": 2,
  "errorMessage": "answer success"
}
