    # 设备日志内容上报

    发起端：终端设备

    鉴权方式：参照《1.2.1 签名方式说明》,注意 header 中增加 uploadFile=true，签名param=“”，附件和相关参数不处理。

    # head 示例：

    HTTP Method

    POST

    # 请求类型

    form/data

    # URL

    /device-monitor/v2/log/all

    请求参数说明

    参数名 参数类型 参数说明 是否必填
    deviceCode String 设备唯一ID 是
    upTime Long 日志时间戳 是
    requestId String 通知消息得唯一标识 是
    logType Integer 默认 0 是
    file MultipartFile 上送文本文件的二进制流(必须 txt、log 格式压缩成zip、tar.gz 格式上传、小于 10m) 是

    # 请求参数示例

    响应结果

    参数名 参数类型 参数说明 是否必填
    retcode int 返回状态码 是
    msg String 返回信息 是
    data JSON Object 返回业务数据 是

    响应示例

    {
  "retcode": 0,
  "msg": "success",
  "data": null
}
