    # 设备事件上报

    上行

    发起方：客户端/服务端

    协 议 ：MQTT

    method：event-report

    上行 Topic：/topic/pub/event/{AppKey}/{clientId}

    下行 Topic：/topic/sub/event/{AppKey}/{clientId}

    # header 信息

    header 信息包含所有请求必须的通用信息

    # 示例

    {
  "header": {
    "traceId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "payloadVersion": 1,
    "brand": "KT",
    "timestamp": 1234567890123,
    "method": "event-report"
  }
}

    # 请求参数说明
    参数名 参数类型 参数说明 是否必填
    deviceCode String 设备唯一ID 是
    events JSON Object Array 设备事件 是
    └name String 属性名 是
    └value String 属性值 是
    └data String 视频/图片地址 否

    请求参数示例

    {
  "header": {
    "traceId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "payloadVersion": 1,
    "brand": "KT",
    "method": "event-report",
    "timestamp": 1234567890123
  },
  "payload": {
    "deviceCode": "12122",
    "events": [
      {
        "name": "eventName",
        "value": "eventValue",
        "data": "http://xxx.xxx.com/xxx.jpg"
      },
      {
        "name": "eventName",
        "value": "eventValue",
        "data": ""
      }
    ]
  }
}

    参数名 参数类型 参数说明 是否必填

    msg String 返回信息 是
