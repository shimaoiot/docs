    # 设备信息上报

    上行

    发起方：客户端

    协议：MQTT

    method：device-info

    上行 Topic：/topic/pub/device/{AppKey}/{clientId}

    下行 Topic：/topic/sub/device/{AppKey}/{clientId}

    # header 信息

    header 信息包含所有请求必须的通用信息

    # 示例

    {
  "header": {
    "traceId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "payloadVersion": 1,
    "brand": "KT",
    "timestamp": 1234567890123,
    "method": "device-info"
  }
}

    # 请求参数说明

    名称 类型 必填? 取值范围 说明 备注
    appKey String 必填 最大 100 位 应用唯一标识（注意 K 大写）
    deviceSn String 必填 最大 100 位 终端 ID
    upTime Long 必填 最大 13 位数字 终端上送时间（精确到秒）
    totalStorage double 0~double 最大 总存储 单位 mb
    appVersion String 最大 100 位 应用版本号
    romVersion String 最大 100 位 rom 版本号
    osVersion String 最大 100 位 系统版本号
    width double 0~999999 屏幕分辨率宽
    height double 0~999999 屏幕分辨率高
    deviceType String 必填 最大 100 位 设备类型描述 任意值 无枚举
    deviceModel String 必填 最大 100 位 设备型号 任意值 无枚举
    remark String 最大 100 位 备注
