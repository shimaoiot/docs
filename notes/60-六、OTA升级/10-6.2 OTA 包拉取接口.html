    # OTA 包拉取接口

    发起端：终端设备

    **鉴权方式：参照 Simolink - 设备注册 - 签名方式说明

    HTTP Method POST

    **请求类型:**application/json

    各个环境及接口地址

    测试环境 domain：http://xtest-api.shimaoiot.com

    UAT环境 domain：https://uat-api.shimaoiot.com

    生产环境 domain ：https://x-api.shimaoiot.com

    URL ：${domain}/api/simolink/ota-service/v2/checkUpdate

    # 请求参数说明

    参数名 参数类型 参数说明 是否必填
    deviceAppkey String 设备 appkey 是
    deviceCode String 设备code 是
    osType String 操作系统(Android、iOS、 Linux、Windows) 是
    version String 当前版本号 是
    targetVersion String 目标版本号 否
    timestamp long 请求时间戳（10 位， 精确到秒） 是

    # 请求示例

    {
  "deviceAppkey": "sddsffvbdfgsdfwergfghfgwed",
  "deviceCode": "sdbfghcvdfgsfsd",
  "osType": "Android",
  "version": "20",
  "targetVersion": "22",
  "timestamp": 1234567891
}

    # 响应结果

    参数名 参数类型 参数说明 是否必填
    retcode int 返回状态码。0 表示成功，非 0 表示失败 是
    msg String 返回信息 是
    data JSON Object 返回业务数据 是
    LdeviceAppkey String 设备 appkey 是
    Lid long 升级包 ID 是
    LdownloadUrl String 升级包下载链接 是
    LfileSize String 升级包大小 是
    Lmd5 String 升级包 md5 值 是
    Lversion String 升级包版本号 是
    LupdateType Int 更新类型，包括（0： 普通更新、1：强制更新:、2：强制静默更新） 是
    Lconfig String 配置信息 否
    LreleaseNote String 版本更新日志 是

    响应示例

    {
  "retcode": 0,
  "msg": "有更新",
  "data": {
    "deviceAppkey": "sdfdgghfgjfgsdsdfwe",
