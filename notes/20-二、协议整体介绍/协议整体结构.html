# header 参数说明

参数名 参数类型 参数说明 是否必填
traceId String 请求唯一 ID，用于跟踪请求，使用 UUID 是
payloadVersion Int Payload 版本号，当前默认为 1 是
brand String 来源平台编号，由物联云平台分配 是
timestamp long 请求时间戳 是
method String 请求方法名 是

traceId 生成规则

系统当前时间（精确到毫秒，13 位），例如：1599531928703设备唯一 ID，例如：ASDFVFWE123

去除”-”符号的 UUID：b38c86eff97b41c7a87e91b0279a1ba4 UUID 后七位：79a1ba4

系统当前时间+设备唯一 ID+UUID 后七位：1599531928703ASDFVFWE12379a1ba4 MD5（1599531928703ASDFVFWE12379a1ba4） 转 大 写 ： 7CFF11D7B5E769C865B9307E70991236 traceId：7CFF11D7B5E769C865B9307E70991236

以上协议结构适用于所有业务协议整体结构

# MQTT payload method枚举

枚举值 枚举说明
device-control 设备控制
device-desired 获取设备影子期望值
device-group 设备编组
shadow-report 设备影子上报
event-report 事件上报
device-discovery 设备发现
device-bind 设备绑定
device-unbind 设备解绑
log-all 全量日志上报
device-ota 固件升级
device-strategy 设备策略下发
device-strategy-ack 设备策略下发回调
device-info 设备信息上报
pass-rule-info 通行信息下发
pass-rule-ack 通行信息下发回调
pass-record 通行记录
