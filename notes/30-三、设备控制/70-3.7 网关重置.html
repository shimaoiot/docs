    # 网关重置

    上行/下行

    发起方：客户端/服务端

    协议：MQTT

    method：gateway-reset

    上行 Topic：/topic/pub/device/{AppKey}/{clientId}

    下行 Topic：/topic/sub/device/{AppKey}/{clientId}

    # header 信息

    header 信息包含所有请求必须的通用信息

    # 示例

    {
  "header": {
    "traceId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "payloadVersion": 1,
    "brand": "KT",
    "timestamp": 1234567890123,
    "method": "gateway-reset"
  }
}
