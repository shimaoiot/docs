    # 设备编组下发

    发起方：服务端

    协议：MQTT

    method：device-group

    下行 Topic：/topic/sub/group/{AppKey}/{clientId}

    # header 信息

    header 信息包含所有请求必须的通用信息

    # 示例

    {
  "header": {
    "traceId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "payloadVersion": 1,
    "brand": "KT",
    "timestamp": 1234567890123,
    "method": "device-group"
  }
}

    # 请求参数说明

    参数名 参数类型 参数说明 是否必填
    groups JSON Object Array 控制的设备 ID 列表 是
    └groupDeviceCode String 设备分组唯一ID 是
    └groupDeviceName String 设备分组名称 是
    └group String 设备分组模型类型 是
    └devices JSON Object Array 需改变的设备属性 是
    └└deviceCode String 设备唯一ID 是
    └└deviceName String 设备名称 是

    # 请求参数示例

    {
  "header": {
    "traceId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "payloadVersion": 1,
    "brand": "KT",
    "method": "device-group",
    "timestamp": 1234567890123
  },
  "payload": {
    "groups": [
      {
        "groupDeviceCode": "1000",
        "groupDeviceName": "射灯组",
        "groupDeviceType": "light",
        "devices": [
          {
            "deviceCode": "1001",
            "deviceName": "射灯"
          },
          {
            "deviceCode": "1002",
            "deviceName": "射灯"
          }
        ]
      },
      {
        "groupDeviceCode": "2000",
        "groupDeviceName": "筒灯组",
        "groupDeviceType": "light",
        "devices": [
          {
            "deviceCode": "2001",
            "deviceName": "筒灯"
          },
          {
            "deviceCode": "2002",
            "deviceName": "筒灯"
          }
        ]
      }
    ]
  }
}
