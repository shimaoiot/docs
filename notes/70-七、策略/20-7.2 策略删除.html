    # 策略删除

    发起方:服务端

    协议类型:mqtt

    method:device-strategy-delete

    下行 Topic：/topic/sub/strategy/{AppKey}/{clientId}

    # header 信息

    header 信息包含所有请求必须的通用信息

    # 示例

    {
  "header": {
    "traceId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "payloadVersion": 1,
    "brand": "KT",
    "timestamp": 1234567890123,
    "method": "device-strategy-delete"
  }
}

    # 请求参数说明

    参数名 参数类型 参数说明 是否必填
    reqId String 请求ID**（接收方可根据 reqId** 作去重处理） 是
    data object 数据 body 是

    # Data

    参数字段 类型 子字段 子字段类型 说明
    entityStrategys List 实体策略列表
    └strategyId Long 是 策略ID
    └strategyName String 是 策略名称
    └parentStrategyId String 否 父策略 ID
    └parentStrategyName Long 否 父策略名称

    请求参数示例

    {
  "header": {
    "traceId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "payloadVersion": 1,
    "brand": "KT",
    "timestamp": 1234567890123
  },
  "payload": {
    "method": "device-strategy",
    "data": {
      "entityStrategys": [
        {
          "strategyId": 100100,
          "strategyName": "睡眠模式",
          "parentStrategyId": 1001012,
          "parentStrategyName": "夏季模式"
        },
        {
          "strategyId": 100101,
          "strategyName": "制冷模式",
          "parentStrategyId": 1001012,
          "parentStrategyName": "夏季模式"
        }
      ]
    }
  }
}
