    # 策略控制

    发起方:服务端

    协议类型:mqtt

    method:strategy-control

    下行 Topic：/topic/sub/strategy/{AppKey}/{clientId}

    # header 信息

    header 信息包含所有请求必须的通用信息

    # 示例

    {
  "header": {
    "traceId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "payloadVersion": 1,
    "brand": "KT",
    "timestamp": 1234567890123,
    "method": "strategy-control"
  }
}

    # 请求参数说明

    参数名 参数类型 参数说明 是否必填
    strategies JSON Array 设备控制列表 是
    └strategyId String 策略编号 是
    └strategyTag String 策略标签： scence：场景link：联动 是

    请求参数示例

    {
  "strategies": [
    {
      "strategyId": 100101,
      "strategyTag": "scence"
    },
    {
      "strategyId": 100102,
      "strategyTag": "link"
    }
  ]
}
