    # 策略下发

    发起方:服务端

    协议类型:mqtt

    method:device-strategy

    下行 Topic：/topic/sub/strategy/{AppKey}/{clientId}

    # header 信息

    header 信息包含所有请求必须的通用信息

    # 示例
    {
  "header": {
    "traceId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "payloadVersion": 1,
    "brand": "KT",
    "timestamp": 1234567890123,
    "method": "device-strategy"
  }
}

    # 请求参数说明
    参数名 参数类型 参数说明 是否必填
    reqId String 请求ID**（接收方可根据 reqId** 作去重处理） 是
    data object 数据 body 是

    Data

    参数字段 类型 子字段 子字段类型 必须 说明
    entityStrategys List 实体策略列表
    └strategyId Long 是 策略ID
    └strategyName String 是 策略名称
    └thirdStrategyId String 否 第三方厂商策略 ID
    └parentStrategyId Long 否 父策略 ID
    └parentStrategyName String 否 父策略名称
    └strategyOrder Long 否 策略执行顺序、默认 99，数值越小，越先执行
    └strategyTag String 否 策略标签
    └conditions:List<condition> condition id Long 否 触发条件 id
    deviceCode String 是 设备编码
    deviceName String 是 设备名称
    deviceTypeCode String 是 设备类型编码
    deviceModelVersion String 否 设备模型编码
    attrCode String 是 设备属性编号
    operator String 否 操作符-数字（ eq:等于lt：小于mt：大于 le：小于等于me：大于等于）
    numberValue Double 否 对比操作值 operator 不为空时 该值不能为空(数值类型对应字段)
    due_to String 否 原始值
    change_to String 否 目标值（非数值类型对应字段）
    groupFlag Integer 是 同组标识 （1，2，3） 数字代码相同为同一组条件
    └actions:List<action> action id Long 是 执行动作ID
    deviceCode String 否 设备编码（actionType:1 下该字段必填）
    deviceName String 是 设备名称
    deviceTypeCode String 是 设备类型编码
    deviceModelVersion String 否 设备模型版本
    └timers:List<timer> timer id Long 否 时间触发条件 ID
    timerType Integer 是 定时器类型 ( 0：永远开启 ; 1:定时开启 )
    beginTime Long 否 有效开始日期[yyyy-MM-dd] （timerType:1 下该字段必填）【目前该字段可以不用关心】
    endTime Long 否 有效结束日期[yyyy-MM-dd] （timerType:1 下该字段必填）【目前该字段可以不用关心】
    effectBeginTime Long 否 生效开始时间（HH:mm）【目前该字段可以不用关心】
    effectEndTime Long 否 生效结束时间（HH:mm）【目前该字段可以不用关心】
    month String 否 执行月份 默认 ：all（所有月份）;/5(每5月)执行;5(5月)执 行 （timerType:1 下该字段必填，取值范围 1-12）
    day String 否 执行星期（timerType:1 下该字段必填，取值范围 1-7）
    time String 否 执行时间 默认 ：all（所有时间）;/5(每 5 小时)执行;5(5 小时)执行（timerType:1 下该字段必填，取值范围 0-23）
    minute String 否 执行分钟：al（l 所有分钟）;/5(每 5 分钟) 执行;5(5 分钟) 执行 （timerType:1 下该字段必填， 取值范围 0-59）
    second String 否 执行秒：all（所有秒）;/5(每 5 秒 ) 执 行 ;5(5 秒 ) 执行 （timerType:1 下该字段必填，取值范围 0-59）
    groupStrategys List 否 分组策略列表
    └strategyId Long 是 策略编号
    └strategyName String 是 策略名称
    └parentStrategyId Long 否 父策略 ID
    └parentStrategyName String 否 父策略名称
    └strategyOrder Long 否 策略执行顺序、默认 99，数值越小，越先执行
    └strategyTag String 否 策略标签
