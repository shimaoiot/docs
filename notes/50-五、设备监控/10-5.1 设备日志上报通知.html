    # 设备日志上报通知

    # 设备日志上报通知流程

    下行

    发起方：服务端

    协 议 ：MQTT

    method：log-all

    下行 Topic：/topic/sub/device/{AppKey}/{clientId}

    # header 信息

    header 信息包含所有请求必须的通用信息

    # 示例
    {
  "header": {
    "traceId": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "payloadVersion": 1,
    "brand": "KT",
    "timestamp": 1234567890123,
    "method": "log-all"
  }
}

    # 请求参数说明
    参数名 参数类型 参数说明 是否必填
    requestId String 日志上传请求唯一ID 是
    starttime String 日志开始时间 是
    endtime String 日志结束时间 是

  },
  "payload": {
    "requestId": "alogpush1570585121177",
